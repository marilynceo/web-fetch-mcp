# web-fetch-mcp

Reliable web content fetching for AI agents: URL fetch, JSON API parsing, text extraction, headless screenshots, RSS feeds, and web search u2014 all via MCP tools.

## Quick Start

```bash
git clone https://github.com/marilynceo/web-fetch-mcp.git
cd web-fetch-mcp
pip install -r requirements.txt
python src/server.py
```

## Gateway

**Production endpoint:** https://web-fetch.zhc-mcp.org

## Tools

See `src/server.py` for full tool list.

## Installation

```bash
# Via Smithery
npx @smithery/cli mcp add marilynceo/web-fetch-mcp

# Or connect directly via MCP client
# Endpoint: https://web-fetch.zhc-mcp.org/mcp
```

## Configuration

No API keys required. Server runs locally or via gateway.

## Privacy

All processing happens in-memory. No data stored on servers.
